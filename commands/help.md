Show the user the copilot-cli skill reference, then list these available slash commands:

| Command | What it does | Example |
|---------|-------------|---------|
| `/cofluent:auto` | Activate autonomous Copilot CLI mode for the session | `/cofluent:auto` |
| `/cofluent:verify` | Verify Copilot CLI is installed and authenticated | `/cofluent:verify` |
| `/cofluent:ask` | Ask Copilot any question | `/cofluent:ask how do I reverse a string in bash?` |
| `/cofluent:suggest` | Get a shell command suggestion | `/cofluent:suggest delete all node_modules recursively` |
| `/cofluent:explain` | Explain a command or snippet | `/cofluent:explain git rebase -i HEAD~3` |
| `/cofluent:fix` | Fix an error or bug | `/cofluent:fix permission denied running npm install` |
| `/cofluent:review` | Review staged diff or a file | `/cofluent:review` or `/cofluent:review src/index.ts` |
| `/cofluent:help` | Show this reference | `/cofluent:help` |
